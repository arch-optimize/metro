# Project Metro

This project aims to optimize metro train operations by better matching service supply with passenger demand. It improves scheduling efficiency, reduces energy consumption and operational costs, and enhances overall service reliability using advanced data analysis and simulation techniques.

## Requirements

- [Conda](https://docs.conda.io/projects/conda/en/latest/user-guide/install/)
- Python 3.11
- `requirements.txt` (included in this repository)

---

## Installation

### 1. Install Conda

If you don’t have Conda installed, download and install **Anaconda** or **Miniconda**:

- [Anaconda](https://www.anaconda.com/products/distribution)
- [Miniconda](https://docs.conda.io/en/latest/miniconda.html)

---

### 2. Create Conda Environment
```bash
conda create -n myenv python=3.11 -y
```
### 3. Activate Conda Environment
```bash
conda activate myenv
```

### 4. Install Dependencies
```bash
pip install -r requirements.txt
```

## Usage
Run the application:
```bash
python app.py ()
```
